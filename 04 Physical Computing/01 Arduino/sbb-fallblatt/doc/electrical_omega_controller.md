# Electrical details Omega controller

## General

* Protocol: RS422

## Pinout

| Pin      | Function | Comment                |
|----------|----------|------------------------|
| Data +   | Tx +     | + data to controller   |
| Data -   | Tx -     | - data to controller   |
| RB +     | Rx +     | + data from controller |
| RB -     | Rx -     | - data from controller |
